Sankraman shows relative abundance of a species on weekly basis which can help understanding migration in the species on a very high level.

Download occurrence data for a species from https://www.gbif.org/ in Darwin Core Archive format.

Unzip occurrence.txt from the downloaded zip and select for processing.

Paste the full citation from GBIF download page or from the email notification. It will be displayed in the footer.

Animation of weekly relative abundance can be seen on the map.
